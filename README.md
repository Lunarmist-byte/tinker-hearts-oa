## The Thattikootu Stack

Frontend: v0 (Because I hate Frontend) Sathyam parayalo, CSS center aakan ulla kadi karanam full AI vech thatti kootiyatha. Frontend paniyan ulla budhimuttu backend aalkarkke ariyu. Athukond design-il kuttam parayaruthu, v0-odu poyi choicha mathi.

Backend: The "Kadu" Stack (Hand-crafted by @Lunarmist-byte ft. Chaya)

Frontend-il "div" center aakan patatha aalkar ithu vayichitt karyamilla. Still, for the nerds:

    Python: Because snakes represent your exes perfectly.

    TensorFlow/Keras: Verum if-else ezhuthi maduthappo undakiya CNN Model. Basically, we used a nuclear weapon to kill a mosquito.

    Pandas: Excel sheet-il ninnu ninte vidhi (data) vayikkan. (Because databases are too mainstream).

    TextBlob: Sentiment analysis. Nee happy aano, sad aano, atho verum 'thallu' aano ennu check cheyunnu.

    FLAMES: The Ultimate Framework. Google-um OpenAI-yum polum tholkkunna logic. Science < Astrology < FLAMES.
